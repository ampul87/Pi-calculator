# Pi Network Conversion Calculator

A Python-based calculator to convert Pi Network (PI) to USD and MYR, and vice versa, using real-time data from CoinGecko and exchangerate.host APIs.

## Features
- Convert PI to USD and MYR.
- Convert USD and MYR to PI.
- Real-time price updates using CoinGecko for PI/USD and exchangerate.host for USD/MYR.
- Error handling for invalid inputs and API failures.

## Prerequisites
- Python 3.6+
- `requests` library (`pip install requests`)
